For a set of formulas $F$ let $\mathsf{At}(F)$ denote the set of propositions appearing in $F$. The MusVar inconsistency measure $\mathcal{I}_{mv}$ is then defined as the ratio of the number of the number of propositions appearing in any minimal inconsistent subsets and the total number of propositions: \[ \mathcal{I}_{mv}(\mathcal{K}) = \frac{|\bigcup_{M\in\mathsf{MI}(\mathcal{K})}\mathsf{At}(M)|}{|\mathsf{At}(\mathcal{K})|} \] The MusVar inconsistency measure has been proposed in [Xiao:2012].
