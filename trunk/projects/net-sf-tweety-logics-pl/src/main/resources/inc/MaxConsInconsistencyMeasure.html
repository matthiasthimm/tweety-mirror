Let $\mathsf{MC}(\mathcal{K})$ be the set of maximal consistent subsets of $\mathcal{K}$, i.e., $\mathsf{MC}(\mathcal{K})=\{\mathcal{K}'\subseteq\mathcal{K}\mid \mathcal{K}'\not\models\perp\wedge\forall\mathcal{K}''\supsetneq\mathcal{K}':\mathcal{K}''\models\perp\}$, and let $\mathsf{SC}(\mathcal{K})$ be the set of self-contradictory formulas of $\mathcal{K}$, i.e., $\mathsf{SC}(\mathcal{K})=\{\phi\in\mathcal{K}\mid\phi\models\perp\}$. Then the MaxCons inconsistency measure $\mathcal{I}_{mc}$ is defined as \[ \mathcal{I}_{mc}(\mathcal{K}) = |\mathsf{MC}(\mathcal{K})| + |\mathsf{SC}(\mathcal{K})| - 1 \] The MaxCons inconsistency measure has been discussed in e.g. [Grant:2011].
