A knowledge base $\mathcal{K}$ is called n-consistent if every subset $\mathcal{K}'\subseteq\mathcal{K}$ with $|\mathcal{K}'|=n$ is consistent [#Doder:2010], let $n_{\mathcal{K}}$ be largest number such that $\mathcal{K}$ is $n_{\mathcal{K}}$-consistent. Then the NCons inconsistency measure $\mathcal{I}_{nc}$ is defined as \[ \mathcal{I}_{nc}(\mathcal{K}) = |\mathcal{K}| - n_{\mathcal{K}} \] For a discussion on the above measure see [Doder:2010].
